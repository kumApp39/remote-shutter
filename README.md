# リモートシャッター

2台のiPadでつかうリモートシャッターPWA。1台を「カメラ」(三脚などに固定)、もう1台を「リモコン」にして、リモコン側の**画面のどこをタップしても撮影**できます。特別支援学校での使用を想定し、大きなタッチターゲット・ひらがな表記・シンプルな操作を優先しています。

## つかいかた

1. 2台のiPadを**同じWi-Fi**(またはスマートフォンのテザリング)につなぐ
2. 両方のiPadでアプリを開く(Safariで開いて「ホーム画面に追加」推奨)
3. カメラ側: 「📷 カメラ」をタップ → カメラの許可 → **あいことば(4けたの数字)**が表示される
4. リモコン側: 「🎮 リモコン」をタップ → あいことばを入力 → カメラの映像が全画面に映る
5. リモコンの**画面のどこをタップしてもシャッター**が切れる(フラッシュ+シャッター音)
6. 写真・動画はカメラ側のiPadにたまる → サムネイルをタップ → 「📤 共有・保存」で写真アプリへ保存

## 機能

- 写真 / 動画(録画開始・停止)の切り替え — カメラ側から強制切替も、リモコン側切り替えの禁止も可能
- 連打・誤動作防止: シャッター後1.5秒間はタップ無効(チャタリング防止)
- 撮影フィードバック: 画面フラッシュ+シャッター音
- 画面スリープ防止(Wake Lock)

## しくみ・制約

- 通信は [PeerJS](https://peerjs.com)(WebRTC)によるP2P。接続のきっかけ作り(シグナリング)にのみPeerJSの無料クラウドを使い、映像は端末間を直接流れる
- **2台が同じネットワークにいること**と、**接続時にインターネットに出られること**が必要
- 撮影データは端末の外部サーバーには送信されない
- ページを閉じる/再読み込みすると未保存の写真・動画は消える(保存してから終了)

## 技術構成

- 単一HTML + Vanilla JS(ビルド不要)/ PWA(manifest + Service Worker)
- WebRTC: PeerJS 1.x(同梱)/ カメラ: getUserMedia / 録画: MediaRecorder
